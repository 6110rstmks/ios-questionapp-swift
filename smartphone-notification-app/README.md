# スマートフォン通知アプリ

## 概要
指定した時刻に通知を送信するタイマー通知アプリです。

## 機能
- ⏰ 時刻を指定してタイマーを設定
- 🔔 ローカル通知による時刻通知
- 📝 設定済みタイマーの一覧表示
- ⏱️ 通知までの残り時間表示
- 🗑️ タイマーの削除機能

## 技術スタック
- SwiftUI
- UserNotifications
- Swift Concurrency (async/await)
- MVVM アーキテクチャ

## 必要要件
- iOS 15.0以降
- Xcode 13.0以降

## セットアップ
1. プロジェクトをクローン
2. Xcodeでプロジェクトを開く
3. シミュレータまたは実機でビルド・実行

## 使い方
1. アプリを起動すると通知権限をリクエストされるので許可する
2. 時刻選択のホイールで時・分を設定
3. 「セット」ボタンをタップしてタイマーを追加
4. 設定済みタイマーは一覧に表示される
5. タイマーを削除する場合は左スワイプで削除

## プロジェクト構成
```
smartphone-notification-app/
├── ContentView.swift          # メインのビュー
├── TimerViewModel.swift       # ビューモデル
├── TimerNotification.swift    # タイマー通知のモデル
└── Tests/                     # テストファイル
```
